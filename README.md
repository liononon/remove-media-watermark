# remove-media-watermark
## 介绍
```text
本项目采用SpringBoot框架开发
支持解析内容：
    抖音：单个视频、相册
    快手：单个视频、相册
    微博：单个视频
    网易云音乐：单个视频
```

## 环境要求
```text
maven3
java版本不小于1.8
```

## 编译
```text
项目根目录下运行：
# 安装依赖
mvn install

# 打包
mvn package
```

## 运行
```text
java -jar remove-media-watermark-1.6.0.jar
```
